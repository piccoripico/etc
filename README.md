# 数独 (Sudoku)

`Sudoku.html` をブラウザで直接開くだけで遊べる数独ジェネレーター／ソルバーです。

## 遊び方
1. ファイルマネージャーで `Sudoku.html` をダブルクリックするか、ブラウザの「ファイルを開く」から指定します。
2. 盤面をクリックしてマスを選択し、キーボードのテンキーまたは画面右の数字パッドから 1〜9 を入力します。Backspace/Delete/0 で消去できます。
3. 右上の「新しい盤面を作成」でヒント数やシードを設定して問題を再生成できます。同じシードを入れると同じ盤面が再現されます。
4. タイマーが自動で計測を開始します。盤面が正解状態になったときにのみクリアメッセージと経過時間が表示されます。リセットするとタイマーも再スタートします。
5. 補助機能
   - **完成ハイライト**: ある数字が 9 マスすべて正しく埋まるとその数字のマスが淡い緑色に。
   - **メモ（✏️ ボタン / Enter）**: メモモードで入力すると候補数字を 3x3 の小さな数字として書き込めます。確定入力するとメモは消去されます。
   - **重複警告**: 同じ数字が行・列・ブロックで衝突すると淡い黄色で警告。
   - **Undo/Redo**: ボタンまたは Ctrl+Z / Ctrl+Y（Shift+Ctrl+Z）で直前の操作を戻す／やり直す。
   - **キーボード操作**: 矢印キーでセル移動、Enter でメモモード切替、数字キーで入力、Backspace/Delete/0 で消去。
   - **ヒント / リセット / 解答を表示**: 空マスを 1 つ埋める、初期状態へ戻す、全解答を表示する。
