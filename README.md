# Sudoku (Browser)

`web_index.html` だけで遊べる数独ジェネレーター／ソルバーです。ネットワーク接続やローカルサーバーは不要で、ファイルを直接ブラウザで開くだけで動きます。

## 遊び方

1. `web_index.html` をブラウザで開きます（ダブルクリックで OK）。
2. 盤面のマスをクリックして選択し、キーボードの数字キー／テンキーまたは右側の数字パッドから 1〜9 を入力します。Backspace / Delete / 0 で消去できます。
3. 「新しい盤面を作成」でヒント数やシードを指定して問題を再生成できます。ヒント、リセット、解答表示ボタンも用意しています。
   - 同じシード値を入れると同じ盤面が再現されます（省略時は毎回ランダム）。

開発用途でブラウザのデベロッパーツールを開けば、生成・解答ロジックを含むすべてのコードをその場で確認できます。
