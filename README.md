# Sudoku CLI

Python 製の簡単な数独ジェネレーター／ソルバーです。生成した問題をコマンドライン上で遊んだり、既存の 81 文字パズル文字列を解いたりできます。

## 使い方

Python 3.10 以上を想定しています。

### 新しい問題を遊ぶ

```bash
python sudoku.py --clues 32 --seed 42
```

* `--clues` で初期のヒントの数を指定します（17〜81）。値が少ないほど難しくなります。
* `--seed` を指定すると同じ問題を再現できます（未指定ならランダム）。

起動後のコマンド例:

* `set 1 1 5` : 1 行 1 列に 5 を入力
* `clear 1 1` : 1 行 1 列を消去
* `hint` : 空きマスを 1 つだけ正解で埋める
* `reset` : 盤面を初期状態に戻す
* `solve` : 完全解答を表示して終了
* `quit` : ゲームを終了

### 既存のパズルを解く

81 文字（行方向に 9×9）で、空マスは 0 や `.` を含めたパズル文字列を渡します。

```bash
python sudoku.py --puzzle "530070000600195000098000060800060003400803001700020006060000280000419005000080079" --solve-only
```

`--solve-only` を付けると解答だけを表示して終了し、付けなければその問題で遊べます。
